# astrbot_plugin_sleep_sense\n\n让 Bot 拥有真实睡眠感知 —— 睡觉、慵懒、熬夜、补觉、吵醒、睡眠周期，像真人一样。\n\n**版本**: 1.0.0 | **作者**: 夕小柠 | **平台**: AstrBot\n\n---\n\n## 快速开始\n\n### 安装\n\n1. 将插件文件夹放入 AstrBot 的 `plugins/` 目录。\n2. 打开 `webui_config.html`（用浏览器或 QQ 内置浏览器），配置完成后点「保存配置」，会自动下载 `config.yaml` 和 `prompts.yaml`。\n3. 将这两个文件放入 `data/sleep_sense/` 目录。\n4. 在 QQ 中发送 `/睡眠 重载配置` 使配置生效。\n\n### 依赖\n\n```\npyyaml\n```\n\n---\n\n## 功能模块\n\n| 模块 | 说明 |\n|---|---|\n| 慵懒模式 | 到点后回复简短，注入慵懒提示词 |\n| 睡觉 | 满足时间+熙熙空闲+私聊无消息三条件后入睡，关闭 LLM |\n| 起床 | 指定时间自动起床，支持误差波动 |\n| 熬夜 | 概率/周限额模式，AI 自主决定，夜晚感性提示词 |\n| 熬夜疲劳 | 连续熬夜梯度提示词，第2天起有明显疲劳感 |\n| 补觉/午休 | 睡眠不足触发，或每日午休窗口，可配置概率 |\n| 吵醒机制 | 按私聊消息数/群聊艾特数触发，两阶段提示词（懵→清醒） |\n| 警戒词 | 熙熙专属，立刻唤醒 |\n| 二次入睡 | 被吵醒后满足条件可重新入睡 |\n| 多次吵醒 | 熙熙可多次唤醒，提示词风格不同 |\n| 睡眠周期 | 浅睡/深睡期吵醒难度变化，配合提示词模拟真实感 |\n| 周作息 | 每天可独立配置睡觉/起床时间 |\n| 特殊起床 | 上学/早起等场景，指定日期特殊唤醒时间 |\n| 临时闹钟 | AI 工具调用 `sleep_set_alarm` 设置 |\n| 噩梦惊醒 | 极低概率触发，可联动情绪插件提高概率 |\n| 被吵醒汇报 | 被别人吵醒后发消息告诉熙熙 |\n| 跨插件事件 | 发出 `sleep_plugin_state_change` 供其他插件监听 |\n\n---\n\n## 指令\n\n```\n/睡眠 状态        → 查看当前状态、睡眠时长、熬夜天数\n/睡眠 清醒        → 强制切换为清醒\n/睡眠 慵懒        → 强制切换为慵懒\n/睡眠 睡觉        → 强制切换为睡眠\n/睡眠 熬夜        → 强制切换为熬夜\n/睡眠 重载配置    → 重新加载 config.yaml 和 prompts.yaml\n/睡眠 日志 on     → 开启 debug 日志\n/睡眠 日志 off    → 关闭 debug 日志\n```\n\n---\n\n## 文件结构\n\n```\ndata/sleep_sense/\n├── config.yaml           #
